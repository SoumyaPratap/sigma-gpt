# 🤖 Sigma GPT - AI Powered MERN Chat Application

Sigma GPT is a full-stack AI-powered chat application inspired by modern conversational AI platforms. It provides intelligent assistance through multiple tools including Resume Analysis, DSA Support, and Mock Interview Preparation.

---

## 🌐 Live Demo

🔗 https://sigma-gpt-bay-gamma.vercel.app

---

## ✨ Features

### 🔐 Authentication

- Secure User Registration
- Login System
- JWT Authentication
- Protected Routes

### 🤖 AI Assistant

- AI Powered Chat Interface
- OpenRouter AI Integration
- Context Based Conversations

### 📄 Resume Analyzer

- Resume Review
- Skill Analysis
- Improvement Suggestions

### 💻 DSA Assistant

- Coding Problem Support
- Algorithm Explanations
- Interview Preparation Guidance

### 🎤 Mock Interview

- Interview Practice
- Technical Questions
- Feedback Assistance

- Responsive Design
- Modern UI
- ChatGPT Inspired Layout

## 🚀 Installation

Clone Repository

```bash
git clone https://github.com/SoumyaPratap/sigma-gpt.git
```

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm start
```
